Getting started with SPOIWO

Adding SPOIWO library references to your project

Since SPOIWO is intended to support conversions to multiple spreadsheet-like formats (at the moment supports: XLSX, CSV and basic HTML) we made a decision to split the framework into a core model library and the number of independent nature libraries. This way users who (let's say) wish to benefit from SPOIWO support for CSV are not obliged to import the Apache POI libraries into their project.

The downside of this decision is a necessity to explicitly specify all the number of libraries for all the spreadsheet natures in your project. Therefore in order to set up the SPOIWO functionality for both HTML and XLSX support you'd need to import the following JARs:

  1. spoiwo-core.jar
  2. spoiwo-nature-xlsx.jar
  3. spoiwo-nature-html.jar
