The main aim of the functional testing suite is to ensure that majority of the functionality offered by Apache POI (currently v3.9) is also available via SPOIWO library and in functional way.

Therefore we document the code, by implementing Apache POI functionality from http://poi.apache.org/spreadsheet/quick-guide.html the SPOIWO way and compare the results with the original implementation.
